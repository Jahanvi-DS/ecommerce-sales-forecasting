# Ecommerce Sales Forecasting

*Live Project Demo:* [View Interactive Dashboard](https://ecommerce-sales-forecasting-ml-app.streamlit.app/)

------

An end-to-end time series forecasting web application that compares traditional statistical modeling with modern machine learning approaches. This project implements *ARIMA/SARIMAX* and *Facebook Prophet* to predict future sales horizons and presents the results through an interactive *Streamlit* dashboard.

---

## 🚀 Features
- *Exploratory Data Analysis (EDA):* Visualized historical sales trends and processed sequential patterns.
- *Model Comparison:* Evaluated performance differences between a linear statistical model (ARIMA) and an additive growth model (Facebook Prophet).
- *Interactive Frontend:* Built a clean user interface with Streamlit to display dynamic visual charts and model predictions.
- *Production Deployment:* Hosted seamlessly via Streamlit Community Cloud for permanent public availability.

---

## 🛠️ Tech Stack
- *Language:* Python
- *Time-Series Models:* ARIMA (Statsmodels), Facebook Prophet
- *Dashboard & UI:* Streamlit, Matplotlib
- *Data Wrangling:* Pandas, NumPy
- *Environment:* Google Colab, Streamlit Community Cloud

---

## 📂 Repository Structure

- forecasting.ipynb : Step-by-step notebook containing data loading and model training
- app.py : Main Streamlit web application source code
- requirements.txt : List of dependencies required to run the environment
- README.md : Project documentation and overview
